# Кубок Авесто-2026 — project instructions

## Единственный источник правды: команды

`src/data/seed.ts` — единственное место, откуда сайт получает стартовые
данные (в чистом localStorage, включая режим инкогнито). Любая правка
названия команды, компании, группы или логотипа, сделанная через админку в
браузере, живёт **только** в localStorage той сессии и **не** отражается в
коде/git/деплое, пока её явно не перенесли в `seed.ts`.

Если пользователь просит «сохранить» или «зафиксировать» текущее состояние
команд — не полагаться на память/скриншоты прошлых сообщений. Запросить
точный экспорт из его браузера (см. `MEMORY.md`) и перенести в `seed.ts`
дословно, включая логотипы, где это возможно.

## Session-end logging

Перед завершением сессии, если были значимые изменения:
- Допиши короткую запись в `MEMORY.md` в раздел «Лог изменений»: дата, что
  изменилось, почему (пропускать тривиальные правки типа опечаток).
- Если менялась структура/состав команд в `seed.ts` — обязательно отметь
  это отдельной записью, это самое хрупкое место проекта (см. выше).

Записи короткие — один буллет на изменение, новые сверху.

## Проверка перед сдачей работы

- `npx tsc --noEmit -p tsconfig.app.json` — чисто
- `npx vitest run` — все тесты зелёные
- `npm run build` — сборка проходит
- Визуальная проверка в браузере для UI-изменений (dev-сервер на порту 3002)
